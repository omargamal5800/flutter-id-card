# flutter-id-card

Simple Flutter ID card app with profile image, name, role, and contact info.

## Getting Started

### Requirements
- Flutter SDK

### Run
```bash
flutter pub get
flutter run
```

### Assets
The profile image is loaded from:
```
image/mn.jpg
```

Make sure assets are listed in:
```
pubspec.yaml
```

### Screenshots
Add screenshots to:
```
docs/screenshot.png
```
Then update the README if you want to show them here.
